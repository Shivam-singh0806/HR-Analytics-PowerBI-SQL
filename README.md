# HR Analytics Dashboard | SQL + Power BI

## Project Overview

This project focuses on HR data analysis using SQL Server and Power BI.

The raw HR dataset was imported from Excel into Microsoft SQL Server, where the data was transformed and structured into a proper Star Schema model for better analytical performance and reporting.

The cleaned and modeled data was then connected to Power BI using Power Query, where interactive dashboards and KPIs were created using DAX measures.

Additionally, Row Level Security (RLS) was implemented to provide role-based dashboard access for Senior Managers and Regional Directors.
